 1. Description
 --------------------------------------------------------------------

 With the Enigma Codebook Tool, you can create code sheets with key
 settings for different models of German Enigma cipher machines.
 The sheets contain all necessary information to setup your Enigma.
 The program can create and save a single code sheet, valid for one
 month, or a complete year.

 The program has a clear and simple userinterface to view existing
 and create new sheets.


 --------------------------------------------------------------------
 2. Installation
 --------------------------------------------------------------------
 
 System requirements: Windows 98 or higher, mouse installed.

 To install the program:
 Open with Winzip © and choose install, or extract to empty folder
 and run setup.exe.

 To uninstall:
 Open the configuration screen, choose software, select
 'Enigma Codebook Tool' in the list of programs and click the
 Add/Remove button.


 --------------------------------------------------------------------
 3. What's new on v2.0
 --------------------------------------------------------------------

 - Day order on keysheet reversed, as on original German sheets.
 - Plugs changed to maximum 10 since German Forces used only 10 cables
 - Blocking of unallowed characters in Codename
 - Updated helpfile
 - Landscape printout
 - Frame drawings
